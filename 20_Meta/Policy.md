# P-Reinforce 분류 정책 (Policy)

| 요소 | 가중치 | 설명 |
|------|--------|------|
| w1: Categorization Accuracy | 0.4 | 분류 정확도 |
| w2: Graph Connectivity | 0.35 | 지식 간 연결 밀도 |
| w3: User Satisfaction | 0.25 | 사용자 피드백 반영 |

## 분류 규칙

| 조건 | 행동 |
|------|------|
| 기존 분류 유사도 ≥ 85% | 기존 폴더에 배치 |
| 기존 분류 유사도 < 85% | 신규 상위 개념 도출 → 새 폴더 생성 |
| 폴더 내 파일 > 12개 | 하위 카테고리 세분화 제안 |
