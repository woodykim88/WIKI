## 리뷰 AI 인사이트 Pro 용어 정리

## AI VOC 용어 정리

### 플랜 구조

#### 무료플랜

부정 VOC + 1차 태그를 쿼터 없이 제공.

2차 태그는 분석 버튼 클릭 시에만 제한적으로 제공

미니 상품 분석 리포트 월 1회 발송.

#### 무료 체험 플랜

14일 제공

1회 유료 구독 플랜을 겪었을 경우, 다시 무료체험을 겪을 수 없음 (다만 세일즈팀에 의해 유동적으로 환경이 변할 수는 있음)

체험 시작 시점에 2차 태그 1000건 소급 분석 자동 실행

정규 상품 분석 리포트

7일·14일차 2회 발송 보장.

#### 유료 구독 플랜

부정 VOC 리포트 + 1차 태그 + 2차 태그 분석(대시보드), 정규 상품 리포트 전체 제공

정규 리포트 주 1회 발송 (전 2주간 내용을 분석)

무료 체험 플랜 종료 후 자동 전환 가능

무료플랜에서 직접 구독 가능

### 분석 기능

#### 부정 VOC

리뷰 중 부정적인 고객 의견을 탐지·분류하는 분석 모든 플랜에서 쿼터 없이 제공되는 기본 기능

위 내용을 기반으로 리뷰 분석 리포트(알림톡/슬랙) 제공

#### 1차 태그

리뷰를 대분류 카테고리로 태깅하는 분석. 무료 플랜 포함 전 플랜에서 쿼터 없이 상시 제공

쇼핑몰 카테고리별로 제작할 수 있는 1차 태그는 제한되어있다. 다만 업체가 요청할 경우, 1차 태그를 수동 제작해줄 수 있다.

1,000건 무료 분석 시, 지난 1년간 수집된 리뷰 중 1,000건만 분석

#### 2차 태그

1차 태그보다 세부적인 분류 태깅. 무료 플랜에서 \[분석 버튼\] 클릭 시 쿼터(1000건) 내에서만 제공 무료 체험 플랜일때는 2차 태그 분석을 1,000개 소급 적용해준다. (미니 상품분석 리포트의 퀄리트 보장을 위함) 유료구독 플랜에서 플랜은 상시 제공.

1,000건 무료 분석 시, 지난 180일간 수집된 리뷰 중 1,000건만 분석

#### 분석 버튼

A 플랜 사용자가 2차 태그 분석을 수동으로 요청하는 버튼. 월간리뷰브리핑 / 개선필요상품 / 브랜드 인식 리포트 3 페이지에 전부 존재하여 요청할 수. ㅣㅆ다. 클릭 시 최신순 리뷰 기준으로 소급 분석 실행 비용 통제 목적으로 설계된 의도적 마찰 요소로, 해당 버튼을 누르지 않았을 경우 발송되는 리포트에는 2차 태그 관련된 부분은 블러처리가 되어 나올 수 있다.

#### 소급 적용

버튼 클릭 또는 플랜 전환 시점 기준으로 과거 리뷰를 거슬러 올라가 분석하는 방식. 소급 범위는 최대 수개월치, 그 안에서 최근 1000건 선택.

### 리포트

#### 정규 상품분석 리포트

유료 구독 플랜 대상 주 1회 발송되는 리포트. 구독 시작일 기준 +7일 주기로 발송되며, 기존 14일에 분석된 내용을 보내준다. 2차 태그 분석 여부에 따라 콘텐츠 구성이 달라지며, 2차 태그 미분석 시 해당 영역 블러 처리.

#### 미니 상품분석 리포트

무료체험 플랜 대상 월 1회 발송되는 축약 리포 전월 데이터 기준 부정 VOC + 1차 태그 중심으로 구성 2차 태그 영역은 블러 처리하여 무료 체험 플랜으로 업그레이드 유도

#### 리뷰 분석 리포트

무료 플랜 대상 주 1회 / 매일 발송되는 리포트 슬랙 및 알림톡 형태로 현재 수집된 리뷰에 대한 정보 및 부정 리뷰 현황 확인 가능 부정 VOC 위주로 으로 구성

#### 블러 처리

2차 태그 분석이 없는 상태에서 해당 차트·데이터 영역을 흐릿하게 가리는 UI 처리. 구현방식은 저품질로 보이지 않도록 디폴트 데이터를 프론트엔드에서 삽입 후 블러 적용 구독상태와 리포트가 작성된 날짜기준(대시보드에서는 날짜필터의 기준)으로 2차태그 분석 결과가 있냐,없냐를 조합해서 블러처리 유무 결정 구독 상태는 백엔드에서 보내주고, 2차태그 분석결과 유무는 프론트에서 판단

### 데이터 & 쿼터

#### 쿼터 (1000건)

A 플랜에서 분석 버튼 1회 클릭 시 소비되는 최대 분석 건수 제한. 리뷰 수가 1000건 미만이면 실제 수만큼 소비. 매달 1일에 1회 클릭 가능

#### 빈 구간

분석 버튼을 여러 번 클릭할 때 분석되지 않은 리뷰 구간이 중간에 생기는 현상. 무료 플랜의 의도적 불편함으로 설계됨. 리포트에서 해당 구간은 공백으로 표시.

### 전환 플로우

#### 태그 분석 활성화 플로우

업체가 서비스 내에서 AI VOC 기능을 켜는 시점. 이 시점부터 1차 태그 분석 시작. 유료 구독 플랜 활성화와 별개로 수동 활성화 / 비활성화 가능 활성화하지 않으면 1차 태그 뿐만 아니라 2차 태그 역시 분석이 실행되지 않음.

#### Edge 케이스 테스트

메인 플로우를 벗어난 예외적 사용 시나리오. 예: C→A 다운그레이드 후 재구독, AI VOC Pro 먼저 구독 후 태그 기능 나중에 활성화, 쿼터 초과 후 재클릭 등.
