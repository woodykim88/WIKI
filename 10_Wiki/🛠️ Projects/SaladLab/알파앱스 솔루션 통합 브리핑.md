## 📖 핵심 도메인별 상세 (Domain Deep-Dive)

### 📖-1. 알파리뷰 핵심 기능

**리뷰 수집 및 작성 최적화:**
- **간편 프로세스:** 단 1초 만에 리뷰 작성 환경 제공 → 작성률 최대 22.8% 달성 사례
- **알림톡 시스템:** 발송 채널 및 시간 설정 가능, 자동 리마인더 지원
- **리뷰 유형:** 텍스트, 포토, 설문 리뷰, 키워드 리뷰, 스태프 리뷰
- **SNS 연동:** 인스타그램 후기 연동, SNS 리뷰 챌린지를 통해 외부 리뷰를 자사몰로 유입

**리뷰 관리 및 전시:**
- **위젯 시스템:** 20개 이상 위젯(롤링 위젯, 브리핑 위젯, 세일즈 티커 등) 활용
- **상품 간 리뷰 연결:** 동일/유사 상품군 사이 리뷰 공유 → 리뷰 부족 상품 전환율 보완
- **AI 리뷰 분석:** NLP 기반 키워드 분석 → 긍정/부정 요소 파악 → VOC 분석 리포트 제공
- **베스트 리뷰 선정:** 상단 고정 또는 별도 혜택 지급 가능

### 📖-3. 알파업셀 & 알파푸시 & 알파씨드

- **알파업셀(Alpha Upsell):** 장바구니 분석 기반 연관 상품 추천 → 객단가 제고. 14일 체험 후 유료 전환 (최소 39,000원/월)
- **알파푸시(Alpha Push):** 실시간 푸시 알림 기반 행동 유도. **무료 구독 요금제 제공** (구독료 0원, 발송 비용 종량제).
  - **무료 버전 기능 제한:** 온사이트 캠페인 1개, 자동화 메시지 캠페인 1개까지만 생성 가능
  - **무료 버전 불가 기능:** 초스피드 회원가입, 배너 관리, 탐색 유도 장치, 선물하기, 첫 구매 할인
  - **유료 버전:** 월 39,000원 (기능 제한 없음)
  - 발송 비용: 알림톡·문자 발송 시 건당 비용 익월 청구 (무료 버전에도 동일 적용)
- **알파씨드(Alpha Seed):** 체험단 마케팅 솔루션 → 신제품 출시·초기 창업 시 포토 리뷰 신속 확보

### 📖-4. 운영 정책 및 주요 유의사항

**포인트 지급 설정:**
- 리뷰 작성 시 자동 지급 설정 가능
- ⚠️ 상품 금액이 낮거나, 비회원/Npay/Zpay 구매 건은 포인트 지급 제한 가능
- 스태프 리뷰 작성 시 네이버 쇼핑 연동 주의사항 준수 필요 (구매 후기 개수 연동 가능)

**시스템 관리:**
- ⚠️ **디자인 리뉴얼 시:** 알파리뷰 위젯 재설치 작업 필수
- ⚠️ **대형 이벤트 전:** 사전에 알파리뷰 측에 공유하여 시스템 안정성 확보
- **해지 후 데이터:** 서비스 제공 중 작성된 리뷰 데이터 처리 정책 사전 확인 필요
