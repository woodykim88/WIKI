# 알파리뷰 위젯 디자인 및 노출 설정

본 문서는 쇼핑몰 고객에게 수집된 리뷰를 보여주는 프론트엔드 환경인 "위젯(Widget)"의 공통 디자인 설정부터, 특정 페이지 노출/제한 및 라벨 지정 방법에 대한 지식동기화 문서입니다.

## 1. 지식 메타데이터
- **문서 분류:** 프론트엔드 노출 및 디자인 가이드
- **작성 일자:** 2026-04-19
- **관련 프로덕트:** 알파리뷰(Alpha Review), 알파업셀(Alpha Upsell)
- **핵심 키워드:** 공통디자인, 마스킹, 구매인증마크, 위젯OFF, 북마크, 리뷰지정, 상하단고정

---

## 2. 공통 디자인 설정 (Global Configuration)

알파앱스의 위젯은 개별 위젯 세팅도 가능하지만, 전체 위젯 화면을 관통하는 통일된 공통 디자인을 제어할 수 있습니다. 설정 변경 후 반영까지 최대 15분 소요될 수 있으며, 미반영 시 브라우저 캐시 삭제가 필요합니다.

### 2.1. 색상 및 폰트
- 어드민 내비게이션 `[디자인 설정 > 공통 디자인]` 에서 브랜드의 키 컬러와 폰트를 지정하여 모든 위젯에 동일한 톤앤매너를 적용할 수 있습니다.

### 2.2. 작성자 정보 표시 및 마스킹
- **작성자명 표시:** 일반 고객의 경우 아이디(ID) 또는 이름 중 노출 형식을 선택할 수 있습니다. (비회원은 무조건 '비회원' 텍스트로 노출됨)
- **간편가입 회원:** 카카오/네이버 등 간편 로그인을 통해 작성된 리뷰의 경우는 아이디 대신 해당 텍스트(`카카오 로그인` 등)로 표기됩니다.
- **마스킹(별표 처리):** 이름이나 아이디의 노출 수위를 조절할 수 있습니다 (기본값 설정됨). 예: 뒤에서부터 2자리를 가려 "홍**" 등으로 표시 가능하며, 중간 마스킹(홍*동) 방식은 지원하지 않습니다.

### 2.3. 구매 인증 뱃지 (실제 구매 여부)
- 리뷰 데이터에 호스팅사의 '실제 주문 데이터'가 연동 결합되어 있는 경우 "구매 인증" 아이콘이 생성됩니다. 관리자가 임의로 엑셀로 기입한 서포터즈 리뷰나 스태프 리뷰에는 해당 마크가 달리지 않아, 고객에게 신뢰도를 부여하는 장치로 활용됩니다.

## 3. 특정 리뷰 지정 노출 (북마크/라벨링 기능)

상품 상세페이지가 아닌 메인페이지, 기획전(프로모션) 페이지에서 특정 상품이나 선별된 리뷰만 모아서 강조하고 싶을 때 사용합니다.

### 3.1. 위젯 게시 기준 방식의 이해
- **위젯이 설치된 상품 페이지의 리뷰 (기본값):** 고객이 열람 중인 상세페이지의 고유 상품 정보에 맞는 리뷰만 필터링하여 자동으로 보여줍니다. (상세페이지 권장)
- **리뷰 직접 선택:** 관리자가 지정한 상품, 카테고리, 혹은 '북마크'된 리뷰만 보여줍니다. 상세페이지에 적용하면 다른 상품 페이지를 열어도 무조건 지정된 리뷰만 나오게 되므로, **게시판이나 메인페이지**에서만 사용을 권장합니다.

### 3.2. 북마크(특수 위젯 구성) 활용
- 베스트 리뷰 위젯, 스마트픽, 스토리 리뷰 등의 "특수 위젯"은 어드민 `[리뷰 목록]`에서 해당 위젯 이름과 동일한 이름으로 **북마크 라벨**을 부착한 리뷰만 수집하여 게시합니다.
- 복수의 리뷰에 라벨을 부착할 경우, 일괄 처리가 불가능하므로 각각 개별적으로 세팅해야 합니다.

## 4. 리뷰 노출 제어 (위젯 OFF 및 고정/추천)

### 4.1. 특정 페이지에만 위젯 OFF (블랙리스트)
비밀 결제 링크, B2B 도매 전용 상품, 정기 구독 전용 상품, 부록 상품 등 일반 리뷰나 알파업셀(추천 기능)이 노출되면 영업 밸런스를 해치는 상품을 필터링합니다.
- `[추천 > 특정상품에 위젯 OFF]` 메뉴에서 제외 상자를 구성하면, 해당 상품의 상세페이지 영역에는 어떠한 자동 위젯도 노출되지 않습니다.
- 제외 목록은 상품 이미지 컷에 마우스를 오버하여 즉각 삭제(복구)가 가능합니다. 반영까지 약 5분이 소요됩니다.

### 4.2. 우선 순위 제어 (상하단 고정)
리뷰 목록을 볼 때 작성일이나 랜덤순의 한계를 극복하기 위해, 포토 퀄리티가 우수하거나 고객의 신체 스펙이 매력적으로 드러난 리뷰를 의도적으로 게시 순위 최상단으로 끌어올리거나(상단 고정), 악의가 없으나 사진이 보기 불편한 리뷰를 최하단으로 숨길(내릴) 수 있습니다.
