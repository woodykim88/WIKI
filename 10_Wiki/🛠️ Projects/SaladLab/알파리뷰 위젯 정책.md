# [[알파리뷰 위젯 정책]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> 알파리뷰의 위젯 시스템은 "게시판/미디어/팝업/SNS/상품별/데이터분석" 등 20종 이상의 UI 컴포넌트로 세분화되며, 기본 위젯과 유료 패키지 위젯(브리핑/롤링/세일즈 티커)의 기능 차단을 통해 구독-위젯 종속 관계를 형성한다.

## 📐 도메인 모델 개요 (Domain Model Overview)

| # | 도메인 | 핵심 개념 |
|---|--------|-----------|
| 1 | 게시판형 위젯 | 전체 리뷰 조망 목적. 기본/카페24 2종 |
| 2 | 미디어 중심 위젯 | 비주얼 강조. 포토리뷰/심플포토/스마트픽 3종 |
| 3 | 팝업 위젯 | 즉각 집중. 플로팅/채팅 2종 |
| 4 | 베스트리뷰 위젯 | 어드민 선정작 노출. 베스트리뷰/베스트텍스트 2종 |
| 5 | SNS향 위젯 | 커머스 브랜딩. SNS피드/스토리 2종 |
| 6 | 상품별 위젯 | 상품 탐색+리뷰 동시 제공. 목록/게시판/비교 3종 |
| 7 | 단일 기능/유료 위젯 | 전환율 특화. 평점/브리핑(유료)/롤링(유료)/키워드/세일즈 티커(유료) |
| 8 | 보조 컴포넌트 | 공지사항/커스텀 에디터/리뷰 작성 유도(배너/재방문), 인스타 피드 |

## 📖 핵심 도메인별 상세 (Domain Deep-Dive)

### 📖-1. 게시판형 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 게시판(기본) | 기본 | 리뷰 종류 무관, 모든 리뷰를 일목요연하게 노출 |
| 게시판(카페24) | 기본 | 서비스 초기 카페24 기본 리뷰 게시판 디자인과 유사하게 노출. VoC 기반 생성 |

---

### 📖-2. 미디어 중심 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 포토리뷰 | 기본 | 포토리뷰 하이라이트. 텍스트/작성자/상품 정보 함께 표시 |
| 심플포토 | 기본 | 텍스트/작성자/상품 정보 모두 미노출, **미디어만 그리드 형태**로 표시. 주로 상품 썸네일 하단 설치 |
| 스마트픽 | 특수위젯 | 세로형 숏폼 트렌드 반영. 기본 상태에서 텍스트를 미디어 위에 **오버레이**, 마우스 오버 시 오버레이 제거하여 미디어 선명하게 표시 |

---

### 📖-3. 팝업 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 플로팅 | 특수위젯 | 리뷰를 **팝업 형태**로 강조 |
| 채팅 | 특수위젯 | 리뷰를 **채팅 형식**으로 유저 친화적 노출 |

---

### 📖-4. 베스트리뷰 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 베스트리뷰 | 특수위젯 | 베스트리뷰 선정 결과 노출. 포토리뷰 위젯과 유사 UI |
| 베스트 텍스트리뷰 | 특수위젯 | 양질의 **텍스트 리뷰**를 강조 |

---

### 📖-5. SNS향 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| SNS | 기본 | 인스타 피드 형태로 리뷰 표시. SNS리뷰/소셜리뷰 노출 목적 |
| 스토리 | 특수위젯 | 인스타 스토리 형태. 사이트를 트렌디하게 연출 |

---

### 📖-6. 상품별 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 상품별 리뷰 목록 | 기본 | 상품별 대표 리뷰 표시. 상품 탐색 & 리뷰 탐색 동시 제공 |
| 상품별 리뷰 게시판 | 기본 | 동일 목적, 게시판 형태 UI |
| 상품별 리뷰 비교 | 기본 | 동일 목적, 비교 형태 UI |

---

### 📖-7. 단일 기능/유료 위젯

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 평점 | 기본 | 각 상품별 평점, 리뷰 개수, 설문 응답 요약. 주로 게시판 위젯 상단에 조합 사용 |
| **브리핑 위젯** | **유료 (패키지)** | 고객의 구매 옵션, 설문 답변(사이즈, 체형 등) 한눈에 보여줌 → **구매율 상승, 반품율 감소** 목적 |
| **롤링 위젯** | **유료 (패키지)** | 상세페이지 진입 없이 상품목록에서 평균평점 + 리뷰 개수 + 리뷰 내용 노출. 상품목록 설치가 주목적이나 상세페이지에도 설치 가능 |
| 키워드 위젯 | 특수위젯 | 각 상품의 키워드리뷰 데이터 기반 상품 특징 요약 |
| **세일즈 티커** | **유료 (패키지)** | (상세 정책 미기재, 유료 패키지 상품) |

---

### 📖-8. 보조 컴포넌트

| 위젯명 | 종류 | 목적 |
|---|---|---|
| 공지사항 | 기본 | 리뷰 관련 공지 노출 (주로 리뷰 이벤트, 포인트 정책) |
| 커스텀 에디터 | 기본 | 에디터를 통해 자유롭게 편집. 주로 베스트리뷰 이벤트 안내 이미지 노출 |
| 상품 상세 배너 | 리뷰 작성 유도 | 상세페이지 내 리뷰 작성 유도 배너 |
| 재방문 배너 & 플로팅 버튼 | 리뷰 작성 유도 | 재방문 고객 대상 리뷰 작성 유도 |
| 인스타 피드 | 피드/모듈 | 인스타그램 피드 형태 노출 |
