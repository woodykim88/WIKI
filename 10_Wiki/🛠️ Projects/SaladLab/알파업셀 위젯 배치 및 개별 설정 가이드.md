# 알파업셀 위젯 배치 및 개별 설정 가이드

본 문서는 알파업셀(Alpha Upsell)이 쇼핑몰 내 어디에 배치되는지(장바구니, 구매버튼 등) 각 위치별 특성과, 개별 위젯의 모바일/PC 레이아웃 및 디자인을 제어하는 세부 설정 가이드입니다.

## 2. 주요 위젯 배치 위치 및 특성

알파업셀은 고객의 구매 여정에 따라 최적화된 위치에 다양한 위젯을 노출시킬 수 있습니다.

### 2.1. 상세페이지 진입 ~ 결제 전 단계
- **상품 설명 본문 하단 위젯:** 상품 설명을 끝까지 읽은 고객에게 자연스럽게 다른 대안이나 이탈 방지용 상품을 제안합니다.
- **구매버튼 상단 위젯:** 구매를 결정하고 버튼을 누르기 직전, "함께 사면 좋은 상품(크로스셀링)"을 가장 강력하게 유도하는 기본이자 핵심 위젯입니다. 
- **상세페이지 최하단 추천 위젯:** 스크롤이 끝나는 위치에서 쇼핑몰 체류 시간을 늘리기 위해 다른 상품군을 추천합니다.

### 2.2. 장바구니 (퍼널 마지막 단계)
- **장바구니 리스트 상단 위젯:** 결제 직전 장바구니 페이지 상단에 띄우며, 장바구니가 완전히 비어있더라도 AI가 상품을 추천하여 이탈을 방지합니다. 배송비 무료 혜택 컷 달성을 위한 추천 등에 탁월합니다.
- **장바구니 사이드바 / 구매장바구니 버튼 클릭 위젯:** 상품 추가 이벤트가 발생했을 때 즉각 화면 측면 등에 레이어 형태로 반응하여 추가 담기를 유도합니다.

**💡 공통 지원 기능: 퀵뷰(Quick View)**
모든 위젯의 썸네일 우측 하단 "돋보기 아이콘" 클릭 시, 페이지 이동(이탈) 없이 해당 상품의 옵션을 고르고 장바구니에 담을 수 있는 퀵팝업 레이어를 지원합니다.

## 3. 위젯 개별 설정 및 디자인 관리

쇼핑몰 환경에 맞춰 위젯별로 각각 독립적인 세팅(`[위젯 > 개별 설정 > 상세 설정]`)을 진행할 수 있습니다. 반영까지 최대 10~15분이 소요되며, 브라우저 캐시 삭제 후 확인을 권장합니다.

### 3.1. PC / 모바일 분리 설정
- 데스크톱과 모바일 환경의 해상도가 다르기 때문에 **행과 열의 갯수, 최대 표시 페이지 수를 별도 세팅**해야 합니다.
- **내부 컨텐츠 최대 너비:** 쇼핑몰 스킨 가로 사이즈에 맞춰 위젯 크기 폭을 조절합니다.
- **페이지네이션 제어:** '페이지네이션 없음'을 선택하면, 스와이프나 넘기기 기능 없이 첫 로딩된 화면 안의 상품만 노출됩니다.

### 3.2. 상품 노출 규칙 (AI vs 직접 추천)
- **AI 자동 추천 알고리즘:** '함께 많이 구매한 상품' 또는 '많이 본 상품' 등 로직 기준을 수정할 수 있습니다.
- **노출 수량:** 위젯에 띄울 '최소 상품 수'와 '최대 갯수'를 지정해 무분별한 롤링을 제한할 수 있습니다.
- **우선 순위:** 직접 추천(그룹 묶기 등)과 AI 추천을 동시 활성화 시, **직접 추천 세팅 상품이 가장 먼저** 노출됩니다. 만약 둘 다 세팅이 안 되면 위젯 자체가 비노출될 수 있습니다.

### 3.3. 프론트엔드 커스텀 (CSS)
알파앱스 측의 직접 디자인 수정 대행은 스킨 충돌 우려로 지원하지 않으나, HTML/CSS 지식이 있는 경우 자체 커스텀 코드를 삽입할 수 있습니다.
- 특정 요소 크기나 색상 등을 변경할 수 있습니다.
- 썸네일 내부 장바구니 버튼 등을 숨기고 싶을 경우, 크롬 개발자 도구로 클래스 명을 찾아 `display: none;` 코드를 대시보드 CSS 편집 영역에 기재하고 저장하면 즉시 적용됩니다.
