# 리뷰 설문조사 시스템

## 1. 개요

설문조사 시스템은 리뷰 작성 시 구조화된 질문-응답 데이터를 수집하는 기능이다. 쇼핑몰별로 설문 양식을 구성하고, 카테고리별로 다른 설문을 적용할 수 있으며, 고객 프로필 정보와 연계하여 대시보드 분석에 활용된다.

---

## 2. 데이터 모델

### 2.1 SurveyForm (설문 양식)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `title` | String | 설문 양식 제목 |
| `version` | Integer | 설문 버전 |
| `is_disable` | Boolean | 비활성화 여부 |
| `use_all` | Boolean | 전체 상품 적용 여부 |
| `questions` | M2M(SurveyQuestion) | 설문에 포함된 질문 목록 |

### 2.2 SurveyQuestion (설문 질문)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `category` | Choice | 질문 카테고리 (`SurveyQuestion__Category`) |
| `label` | String | 질문 레이블 |
| `title` | String | 질문 제목 |
| `group` | Choice | 질문 그룹 (`SurveyQuestion__Group`) |
| `answer_type` | Choice | 답변 유형 (`SurveyQuestion__AnswerType`) |
| `is_custom` | Boolean | 커스텀 질문 여부 |
| `is_saved_answer` | Boolean | 답변 저장 여부 |

#### 질문 카테고리 (`SurveyQuestion__Category`)

| 카테고리 |
|----------|
| 공통 |
| 패션의류 |
| 반려동물 |
| 식품 |
| 뷰티 |
| 출산/유아동 |
| 가구/인테리어 |

#### 질문 그룹 (`SurveyQuestion__Group`)

| 그룹 | 설명 |
|------|------|
| 고객정보 | 고객 프로필 관련 질문 |
| 배송서비스 | 배송 만족도 관련 질문 |
| 상품퀄리티 | 상품 품질 관련 질문 |

#### 답변 유형 (`SurveyQuestion__AnswerType`)

| 유형 | 설명 |
|------|------|
| 단답형 | 텍스트 자유 입력 |
| 옵션형 | 단일 선택 (라디오 버튼) |
| 목록형 | 드롭다운 목록 선택 |
| 체크형 | 다중 선택 (체크박스) |

### 2.3 SurveyQuestionValue (질문 선택지)

| 필드 | 타입 | 설명 |
|------|------|------|
| `survey_question` | FK(SurveyQuestion) | 질문 참조 |
| `choice` | String | 선택지 텍스트 |

옵션형, 목록형, 체크형 질문에 대한 선택 가능한 값을 정의한다.

### 2.4 SurveyFormQuestionRelation (양식-질문 관계)

| 필드 | 타입 | 설명 |
|------|------|------|
| `survey_form` | FK(SurveyForm) | 설문 양식 참조 |
| `survey_question` | FK(SurveyQuestion) | 질문 참조 |
| `index` | Integer | 질문 표시 순서 |
| `is_visible` | Boolean | 위젯에서 표시 여부 |
| `is_dashboard_only` | Boolean | 대시보드 전용 표시 여부 |

**비즈니스 규칙:** 답변 유형이 `단답형`인 질문은 자동으로 `is_dashboard_only=True`로 설정된다. 단답형 질문은 위젯에서 표시하기 적합하지 않으므로 대시보드에서만 확인 가능하다.

### 2.5 CategorySurveyFormRelation (카테고리-설문 양식 연결)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `category_no` | Integer | 카테고리 번호 |
| `survey_form` | FK(SurveyForm) | 설문 양식 참조 |

특정 상품 카테고리에 맞는 설문 양식을 지정할 수 있다. `use_all=True`인 설문 양식은 카테고리 관계 없이 모든 상품에 적용된다.

### 2.6 ReviewSurveyFormRelation (리뷰-설문 양식 관계)

| 필드 | 타입 | 설명 |
|------|------|------|
| `review` | OneToOne(Review) | 리뷰 참조 (1:1 관계) |
| `survey_form` | FK(SurveyForm) | 설문 양식 참조 |

하나의 리뷰는 하나의 설문 양식과만 연결된다.

### 2.7 ReviewSurveyFormAnswer (설문 응답)

| 필드 | 타입 | 설명 |
|------|------|------|
| `review_survey_form` | FK(ReviewSurveyFormRelation) | 리뷰-설문 관계 참조 |
| `survey_form_question` | FK(SurveyFormQuestionRelation) | 양식-질문 관계 참조 |
| `label` | String | 질문 레이블 (스냅샷) |
| `answer` | String | 응답 텍스트 |
| `keywords` | JSON | 키워드 데이터 |

### 2.8 SurveyAnswerCustomerProfile (고객 프로필 설문 응답)

| 필드 | 타입 | 설명 |
|------|------|------|
| `customer_id` | Integer | 고객 ID |
| `survey_question` | FK(SurveyQuestion) | 질문 참조 |
| `label` | String | 질문 레이블 |
| `answer` | String | 응답 텍스트 |

고객의 설문 응답을 프로필 차원에서 저장하여, 동일 고객이 다른 리뷰에서도 일관된 프로필 정보를 제공할 수 있도록 한다.

## 3. 비즈니스 규칙

### 3.1 설문 적용 범위
- `use_all=True`: 전체 상품에 설문 양식이 적용된다.
- `use_all=False`: `CategorySurveyFormRelation`에 의해 특정 카테고리의 상품에만 적용된다.

### 3.2 질문 표시 규칙
- 단답형 질문은 `is_dashboard_only=True`로 자동 설정되어 위젯에서는 노출되지 않고 대시보드에서만 확인 가능하다.
- `is_visible=False`로 설정된 질문은 위젯과 대시보드 모두에서 숨겨진다.

### 3.3 리뷰-설문 관계
- 리뷰와 설문 양식은 **1:1 관계**(OneToOne)로 구성된다.
- 리뷰의 `is_survey=True` 플래그로 설문 리뷰 여부를 판별한다.
