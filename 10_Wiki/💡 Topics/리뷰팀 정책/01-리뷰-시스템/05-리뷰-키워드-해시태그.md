# 리뷰 키워드/해시태그 시스템

## 1. 개요

키워드 시스템은 리뷰 작성 시 고객이 미리 정의된 키워드를 선택할 수 있도록 하여 구조화된 리뷰 데이터를 수집하는 기능이다. 해시태그 시스템은 카테고리 및 상품 단위로 해시태그를 관리하여 리뷰 검색과 필터링을 지원한다.

---

## 2. 데이터 모델 (키워드)

### 2.1 Keyword (시스템 전역 키워드)

| 필드 | 타입 | 설명 |
|------|------|------|
| `description` | String | 키워드 설명 텍스트 |
| `icon_name` | String | 키워드에 연결된 아이콘 이름 |

시스템 전체에서 공유되는 프리셋 키워드를 정의한다.

### 2.2 KeywordCategory (키워드 카테고리 분류)

| 필드 | 타입 | 설명 |
|------|------|------|
| `category` | Choice | 소분류 카테고리 (`ReviewKeyword__SearchSubSub` choices) |
| `keyword` | FK(Keyword) | 키워드 참조 |
| `display_order` | Integer | 표시 순서 |

### 2.3 KeywordGroup (쇼핑몰별 키워드 그룹)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `title` | String | 그룹 제목 (예: "제품", "가격/서비스") |
| `display_order` | Integer | 표시 순서 |
| `keywords` | M2M(Keyword) | 그룹에 포함된 키워드 목록 |

### 2.4 KeywordGroupRelation (키워드-그룹 관계)

| 필드 | 타입 | 설명 |
|------|------|------|
| `keyword` | FK(Keyword) | 키워드 참조 |
| `keyword_group` | FK(KeywordGroup) | 키워드 그룹 참조 |
| `display_order` | Integer | 그룹 내 표시 순서 |

### 2.5 ReviewKeywordRelation (리뷰-키워드 매핑)

| 필드 | 타입 | 설명 |
|------|------|------|
| `review` | FK(Review) | 리뷰 참조 |
| `keyword` | FK(Keyword) | 키워드 참조 |

- `review`와 `keyword`에 대해 **unique constraint**가 적용되어 동일 리뷰에 동일 키워드 중복 매핑이 불가하다.

---

## 3. 키워드 카테고리 체계

3단계 계층 구조로 키워드를 분류한다.

### 3.1 대분류 (SearchMain)

| 코드 | 이름 |
|------|------|
| 1 | 제품 |
| 2 | 가격/서비스 |

### 3.2 중분류 (SearchSub)

| 카테고리 |
|----------|
| 패션의류 |
| 반려동물 |
| 식품 |
| 뷰티 |
| 디지털 |
| 스포츠/레저 |
| 출산/유아동 |
| 가구/인테리어 |
| 리빙 |

### 3.3 소분류 (SearchSubSub)

`ReviewKeyword__SearchSubSub`에 정의된 **62개 세부 카테고리**로 구성된다.

- 범위: 여성패션 ~ ESG
- 각 소분류는 중분류에 속하며, 키워드 프리셋 할당의 기준이 된다.

---

## 4. 키워드 초기화 흐름

`KeywordGroup__Feature.init()` 메서드는 쇼핑몰에 키워드 그룹이 없을 때 기본 그룹을 자동 생성한다.

### 초기화 순서

```
1. 쇼핑몰의 카테고리에 해당하는 프리셋 키워드 존재 여부 확인
2. 키워드 그룹이 없으면 자동 생성:
   - "제품" 그룹 생성 + 관련 프리셋 키워드 매핑
   - "가격/서비스" 그룹 생성 + 관련 프리셋 키워드 매핑
3. ShopKeywordReview 설정 업데이트 (키워드 리뷰 활성화 상태 반영)
```

## 5. 데이터 모델 (해시태그)

### 5.1 ReviewHashtag (리뷰 해시태그)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `hashtag_type` | Choice | 해시태그 유형 (`ReviewHashtag__HashtagType`: 카테고리 / 상품) |
| `hashtag_text` | String | 해시태그 텍스트 |

### 5.2 ReviewHashtagCategory (해시태그-카테고리 연결)

| 필드 | 타입 | 설명 |
|------|------|------|
| `hashtag` | FK(ReviewHashtag) | 해시태그 참조 |
| `category_id` | Integer | 카테고리 ID |

### 5.3 ReviewHashtagProduct (해시태그-상품 연결)

| 필드 | 타입 | 설명 |
|------|------|------|
| `hashtag` | FK(ReviewHashtag) | 해시태그 참조 |
| `product_id` | Integer | 상품 ID |

## 6. 비즈니스 규칙

### 6.1 키워드 규칙
- 키워드는 시스템 전역으로 정의되며, 쇼핑몰별 그룹으로 묶어 사용한다.
- 리뷰 작성 시 `is_keyword=True`로 설정되면 키워드 리뷰로 분류된다.
- 키워드 그룹은 쇼핑몰 최초 설정 시 자동 초기화된다.

### 6.2 해시태그 규칙
- 해시태그는 카테고리 단위 또는 상품 단위로 설정 가능하다.
- SNS 리뷰에서 해시태그 포함 여부를 검증하는 데 사용된다 (인스타그램 리뷰 인증).
