# 리뷰 챌린지/캠페인

## 1. 개요

챌린지 시스템은 쇼핑몰이 고객에게 SNS 리뷰 작성 미션을 제공하고, 우수 리뷰를 선정하여 추가 보상을 지급하는 캠페인 기능이다. 현재는 SNS(인스타그램) 리뷰만 대상으로 하며, 향후 다른 리뷰 유형으로 확장될 예정이다.

---

## 2. 데이터 모델

### 2.1 ReviewChallenge (챌린지)

| 필드 | 타입 | 설명 |
|------|------|------|
| `shop_id` | Integer | 쇼핑몰 고유 ID |
| `title` | String | 챌린지 제목 |
| `mission_contents` | JSON | 미션 내용 (구조화된 미션 정보) |
| `challenge_status` | Choice | 챌린지 상태 (ACTIVE=200, ENDED=300) |
| `reward_type` | Choice | 보상 유형 (POINT=10, CUSTOM=20) |
| `challenge_reward_point` | Integer | 추가 포인트 보상액 |
| `challenge_reward_custom_text` | String | 업체 설정 커스텀 보상 텍스트 |
| `best_count` | Integer | 베스트 리뷰 선정 수 |
| `target_review_type` | Choice | 대상 리뷰 유형 (기본값: SNS) |
| `challenge_scope` | Choice | 적용 범위 (ALL_PRODUCT=10, PRODUCT=20) |
| `started_at` | DateTime | 챌린지 시작일시 |
| `finished_at` | DateTime | 챌린지 종료일시 |
| `is_deleted` | Boolean | 삭제 여부 (소프트 삭제) |

### 2.2 ReviewChallengeRelation (챌린지-리뷰 관계)

| 필드 | 타입 | 설명 |
|------|------|------|
| `challenge` | FK(ReviewChallenge) | 챌린지 참조 |
| `review` | FK(Review) | 리뷰 참조 |
| `is_best_selected` | Boolean | 베스트 리뷰 선정 여부 |
| `is_best_rewarded` | Boolean | 베스트 보상 지급 완료 여부 |

## 3. 비즈니스 규칙

### 3.1 챌린지 제한
- 쇼핑몰당 **진행 중인 챌린지는 1개만** 허용된다.
- 동시에 여러 챌린지를 운영할 수 없다.

### 3.2 챌린지 생성
- 챌린지 생성 시 **"베스트 SNS리뷰" 특수 라벨**이 자동으로 생성된다.
- 이 라벨은 `Label__LabelType.특수` 유형으로 시스템이 관리한다.

### 3.3 대상 리뷰 유형
- 현재 **SNS 리뷰만 대상**이다 (`target_review_type=SNS`).
- 추후 일반 리뷰, 포토 리뷰 등으로 확장 예정이다.

### 3.4 적용 범위

| 범위 | 코드 | 설명 |
|------|------|------|
| 전체 상품 | ALL_PRODUCT=10 | 쇼핑몰 전체 상품에 대한 리뷰가 챌린지 대상 |
| 특정 상품 | PRODUCT=20 | 지정된 특정 상품에 대한 리뷰만 챌린지 대상 |

### 3.5 상태 전이 규칙

```
생성 → ACTIVE(200) → ENDED(300) → 삭제(소프트)
```

| 전이 | 조건 |
|------|------|
| ACTIVE → ENDED | 진행중(ACTIVE) 상태에서만 종료 가능 |
| ENDED → 삭제 | 종료(ENDED) 상태에서만 삭제 가능 (소프트 삭제) |
| ACTIVE → 삭제 | **불가** (먼저 종료 필요) |

### 3.6 보상 유형

| 유형 | 코드 | 설명 |
|------|------|------|
| 추가 포인트 | POINT=10 | `challenge_reward_point`에 설정된 금액만큼 추가 포인트 지급 |
| 업체 설정 리워드 | CUSTOM=20 | `challenge_reward_custom_text`에 정의된 커스텀 보상 (예: 상품 증정 등) |

### 3.7 베스트 리뷰 선정
- 챌린지 기간 중 관리자가 우수 리뷰를 베스트로 선정한다 (`is_best_selected=True`).
- `best_count`는 선정할 베스트 리뷰의 수를 정의한다.
- 베스트 보상 지급 완료 시 `is_best_rewarded=True`로 업데이트된다.
