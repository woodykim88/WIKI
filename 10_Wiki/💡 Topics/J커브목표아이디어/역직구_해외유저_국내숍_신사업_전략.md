# 역직구 (해외유저-국내숍) 신사업 전략 생각정리

## 📌 핵심 결론 (한 줄)

> 샐러드랩이 "역직구를 우리가 직접 한다"가 아니라, **역직구를 하고 싶은 K-뷰티/K-커머스 셀러가 먼저 찾는 데이터·CRM·리뷰 인프라가 된다**가 가장 자연스럽고 차별화된 포지션이다.

---

## 1. 역직구란?

역직구(Reverse Cross-Border Commerce)는 한국 셀러의 상품을 해외 소비자가 한국 기반 플랫폼에서 직접 주문하고, 한국에서 해외로 발송하는 **온라인 수출 구조**다. 기존 수출이 오프라인(수입상·유통사) 중심이었다면, 역직구는 이를 온라인 직판 형태로 확장한 것.

| 구분 | 수치 | 비고 |
|------|------|------|
| 해외직구 시장 (한국→해외) | 2.2조 → 8.1조 원 | 매우 빠른 성장 |
| 역직구 시장 (해외→한국 구매) | 6천억 → 1.6조 원 | 상대적으로 느린 성장 |
| 역직구 / 해외직구 비율 | 약 **20% 수준** | 구조적 격차 존재 |
| 역직구 성장 둔화 원인 | 외국인 회원가입 복잡성(한국 휴대폰 인증 의존), 글로벌 결제수단 부족 | 중앙은행 분석 |

## 2. 화장품 (K-뷰티)이 역직구 파일럿 카테고리로 유리한 이유

### 강점
- **중량 대비 단가 높음**: 스킨케어·앰플류 100~300g 내외, 높은 객단가 → 소형 패킷 배송 테스트에 적합
- **수요 검증 완료**: K-뷰티 수출 2025년 약 **114억 달러 (사상 최대)**, 두 자릿수 성장 지속 중
- **샐러드랩 자산 연계**: 기존 뷰티 셀러들의 리뷰/재구매 데이터를 기반으로 **"글로벌 히트 가능 SKU 번들"** 선별 가능

### 유의사항
- **부피무게 함정**: 예쁜 화장품 박스가 오히려 부피무게 증가 → 역직구용 외박스 별도 최적화 필요
- **성분·인증·레이블 리스크**: EU 기준 온라인 화장품의 65%가 규정 위반 사례 보고. 국가별 금지/주의 성분 매트릭스 필수
- **CBEC 양허 리스트**: 중국은 CBEC 채널이 있지만, 정부 관리 리스트에 없는 품목은 CBEC 불가 → 일반무역 전환 필요

## 3. 역직구 운영 실전 경험 (mykmarket.com 운영자 Q&A)

> K-뷰티·식품 중심의 미국 역직구 플랫폼 운영 경험자 인터뷰. 아래는 핵심 페인포인트 6개 영역별 요약.

### 3-1. 로지스틱스 (가장 큰 변수)
- 배송비가 **전체 운영비의 50% 이상**을 차지하는 경우도 많음
- DHL, UPS, FedEx, 파스토, 개인 물류사 등 **여러 물류사 병행 운영**이 현실적으로 안정적
- 카페24와 연계된 해외 물류사를 통해 단가 절감 구조 권장
- 기존 역직구 물량이 많은 업체와 함께 묶어 배송하면 초기 단가 경쟁력 확보

### 3-2. 세제·통관 (예측 불가 리스크)
- 정책 변화로 통관이 갑자기 막히는 경우 빈번 (트럼프 정부 시기 지연 사례 다수)
- **식품**: 통관 지연 시 유통기한 문제 → 전량 폐기 리스크 (실제 약 4천만원 폐기 사례)
- 마약 사건 등 항공 연루 시 통관 2개월 이상 묶임
- **물류사 계약 시 통관 지연 리스크에 대한 계약 조건 검토 필수**

### 3-3. 결제 및 환율
- 다양한 글로벌 결제수단 연동 필수: Google Pay, Apple Pay, 글로벌 카드
- **Shopify 기반 플랫폼** 권장: 도난 카드·블랙컨슈머 패턴 감지 기능 내장
- **국가별 결제 테스트 현지 실행 필수**: 국내 테스트만으로는 현지 결제 불가 이슈 미검출
- 환율 버퍼: 정산 리스크 대비 **약 20% 버퍼** 설정 권장 → 환차익 추가 확보 가능

### 3-4. 재무·세무
- 역직구 = **수출** 분류 → 세금 혜택 및 정부 지원 프로그램 다수 존재
- 일반 세무사가 아닌 **역직구/글로벌 커머스 경험 세무사** 협업 강력 권장
- 거래액 커지면 은행 수수료 차이 1% 이상 → 수출 기업 전용 은행 상품 비교 필요

### 3-5. CS
- CS 문의의 **70% 이상이 배송 문의** → 배송 트래킹 자동화로 상당 부분 대체 가능
- 상세페이지 번역, 일반 CS 응대는 번역기 + AI 도구로 처리 가능
- 단, 해외 서류·계약 관련 업무는 전문 인력 필요

### 3-6. 마케팅·가입
- **현지 마케팅사 활용 권장** (국내 마케팅사 대신): 국가별 SNS 정책으로 국내에서 광고 세팅 막히는 경우 다수
- 현지 마케팅사 활용 시 결제 테스트, 배송 점검, 현지 시간대 CS 대응 함께 지원 가능

### 샐러드랩이 따라가면 안 되는 패턴
- **화해식 B2C 플랫폼 직접 운영**: 리뷰 신뢰도·트래픽·한중 브릿지 역량에서 이미 선점 → 정면 승부 불리
- **아비브식 단일 브랜드 직접 육성**: 샐러드랩의 코어 역량(데이터·툴·SaaS)과 거리가 멀고, 조직 모드도 "브랜드 하우스"가 아님

## 6. 샐러드랩이 취할 수 있는 포지션 3가지

### 옵션 A: 직접 B2C 역직구 플랫폼
- **장점**: 기존 셀러 공급망 보유 → 입점 영업 비용 절감, 빠른 SKU 테스트 가능
- **단점**: 해외 트래픽 모객이 전혀 다른 게임. 현재 알파앱스 로드맵과 리소스 충돌 가능성 높음
- **판단**: 리스크 크고, 신중하게 단계적 PoC → MVP → 스핀오프 접근만 유효

### 옵션 B: 기존 셀러의 역직구를 enable하는 B2B SaaS 인프라 ← 가장 유력
- 외국인 회원가입·소셜 로그인 모듈 (휴대폰 인증 없이 가능한 플로우)
- 글로벌 결제 모듈 (PayPal, 카드, 지역별 월렛) + 국내 PG 브릿지
- 다국어 리뷰·추천·온사이트 메시징 (알파리뷰/업셀/푸시 글로벌 버전)
- 역직구 물류/통관/풀필먼트 파트너 연동 SaaS (배송비 계산·관세 안내·통합 대시보드)
- **비즈니스 모델**: 알파앱스 add-on 모듈(Global/역직구 패키지) 구독 + 거래 수수료

### 옵션 C: 글로벌 마켓플레이스와의 파트너링
- 특정 플랫폼(Shopee, 티몰글로벌 등) 내 "한국 셀러 클러스터"를 대상으로 리뷰 수집·번역·추천·CRM·마케팅 자동화를 B2B 파트너로 제공
- 직접 해외 소비자 모객·국제정산 없이 자사 강점만으로 레버리지

## 7. 샐러드랩만의 차별화 포지션 아이디어 (구체 기능 제안)

### 7-1. 역직구 Exportability 스코어
- 카페24/자사몰 셀러의 리뷰·재구매·CS 불만 데이터를 기반으로 "역직구 적합도 스코어" 자동 산출
- 예: "민감성·트러블 케어 제품 중 '피부 진정', '자극 없음' 키워드 + 재구매율 높음 + CS 불만 적음" SKU 우선 추천

### 7-2. 국가별 "문제정의 → SKU 매핑" 엔진
- 국가별 검색어·후기·시장 리포트 + 국내 리뷰 데이터를 연결하여 자동 매핑
- 예: "미국의 여드름 패치 니즈 ↔ 국내 코스알엑스 계열 SKU"
- 단일 브랜드가 아닌 **다수 브랜드 SKU를 니즈 단위로 재조합하는 역직구용 카탈로그**

### 7-3. 역직구 CRM·업셀링 모듈
- 기존 CRM/업셀링 기능을 국가별 "역직구 모드" 템플릿으로 묶어 제공
- 예: 미국 소비자 = "여드름 패치 → 토너 → 세럼" 시퀀스 / 동남아 소비자 = "유분 조절/쿨링" 중심
- 다국어 메시지, 재구매 리마인드, 번들 추천, 리뷰 리마인드 플로우 프리셋 포함

### 7-4. 리뷰 인프라 레이어 (해외 진출 초기 "리뷰 없음" 문제 해결)
- 국내 리뷰를 번역·하이라이트 카드로 변환 → 티몰, 쇼피, 아마존 스토어 등에 내보내는 API 제공
- 국내 + 글로벌 리뷰를 한 대시보드에서 통합 관리

## 8. 실행 순서 (전략 로드맵)

1. **페인포인트 정량화**: 역직구를 이미 시도했거나 강한 의지가 있는 상위 고객사 5~20곳 인터뷰·데이터 분석 → 가입/결제/물류/마케팅 중 어디가 병목인지 확인
2. **MVP 설계**: 가장 큰 페인에 맞춰 "알파앱스 Global/역직구 패키지" 작은 모듈 설계
3. **파일럿 실행**: 뷰티 셀러 대상, 미국·동남아 1~2개 국가, SKU 10~20개, 소형 패킷 번들 구성 → 전환율·해외 매출 uplift·운영 비용 데이터 수집
4. **데이터 기반 스케일업**: 파일럿 결과로 모듈 고도화, 카테고리(K-푸드, 굿즈) 확장, 물류 파트너 정식 API 연동
