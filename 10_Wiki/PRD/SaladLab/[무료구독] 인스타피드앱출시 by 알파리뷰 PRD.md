## 📌 한 줄 통찰 (The Karpathy Summary)

## 📖 핵심 도메인별 상세 (Domain Deep-Dive)

### 📖-1. Problem (문제 해결)

*   **1. 초기 몰 운영자의 성과 인지 부재**:
    *   초기 쇼핑몰 운영자들은 인스타그램, 네이버 블로그 등 소셜 미디어 상에 브랜드 언급이 점진적으로 쌓여가고 있음에도 이를 상품 상세페이지의 "상품 단위 리뷰 자산"으로 연동해 보여주는 구조가 부재하다.
    *   월 주문 50건 내외의 상점은 리뷰 솔루션 비용에 부담을 느끼며 텅 빈 기본 게시판 후기 영역만 방치하고 있다.
*   **2. 잠재적 유료 상점 행동 데이터 유실**:
    *   현재 유료 플랜 중심으로 서비스가 설계되어 있어 초기 상점의 진입 행동 데이터를 확보하지 못하고 있으며, 이로 인해 적절한 성장 타이밍에 맞춰 유료 업셀 넛지를 실행할 기회를 상실하고 있다.

### 📖-2. Appetite (베팅 규모)

*   **베팅 설정**: **6주 배팅** (1순위 2주 + 2순위 2주 + 3순위 2주 배포 테이블 적용)
    *   *1순위 (2주)*: 인스타피드 + 수집 2종 (스태프리뷰, 소셜미디어 가져오기) → 리뷰 축적에 초점.
    *   *2순위 (2주)*: 리뷰 요청 (알림톡 발송 + 카드 등록 조건) + 통합 횟수 제한.
    *   *3순위 (2주)*: 재방문 배너 1종 (주문 연동) + 리뷰 작성 경로 트래킹.
*   **제약 사항**:
    *   기존 알파리뷰의 수집/관리 인프라를 재활용하여 리소스를 축소하고 무료/제한 레이어 설계에 집중한다.
    *   비용 리스크가 큰 포인트 자동 지급 등은 Phase 1 범위에서 배제한다.

### 📖-3. Core Solution (핵심 솔루션)

*   **① 상점 설치 및 온보딩**:
    *   카페24 앱스토어에서 "알파리뷰 – 인스타 피드 연결 무료버전"으로 별도 진입 설치.
    *   온보딩 단계에서 도메인, 카테고리, 규모 및 리뷰 상태를 수집한 후 즉시 소스 연결 화면으로 안내한다.
*   **② 소셜 리뷰 연결 및 피드 구성**:
    *   인스타그램 연동 후 특정 리뷰 포스트 선택 → 상품 연결 → 상품 상세에 그리드/슬라이드 1종 피드 노출.
    *   네이버 블로그 URL 입력 → 대표 이미지/제목/요약 가져오기 기능.
*   **③ 스태프 리뷰 수집**: 스태프/운영자가 직접 작성하는 스태프 리뷰 기능 제공.
*   **④ 리뷰 요청 (기본형)**: 주문 내역에서 최근 30건을 대상으로 첫 번째 리뷰 요청만 알림톡 발송 (이메일/문자는 Phase 2로 이관).
*   **⑤ 리뷰 관리**: 리스트 게시/비게시, 별점/키워드 기본 필터링, 상단 고정(3개 한도) 및 수동 상품 연결(1개 한도) 허용.
*   **⑥ 위젯 노출**: 상세 하단에 인스타 피드 기본 1세트만 노출 (게시판/포토리뷰 등 기본 리뷰 위젯은 Phase 2 범위).
*   **⑦ 자동 설치**: 가능한 경우 위젯 스크립트 자동 삽입 처리, 불가 시 스크립트 가이드 안내 및 유료 설치 대행 옵션 제안.
*   **⑧ 유료 넛지**: 잠긴 설정 화면에 Premium 뱃지 노출, 옵션 변경 시 미리보기는 허용하되 저장 시 업셀 넛지 팝업 노출.

### 📖-4. 리스크 FAQ

*   **Q1. 무료 버전으로 상점이 만족하여 유료 전환하지 않을 가능성**:
    *   A1. 기능의 깊이와 자동화 레벨을 엄격히 고정한다. 통합 쿼터 50건 및 스태프/소셜 연동 수에 한도를 설정하고, 브랜딩 제거 및 자동 댓글, 포인트 자동 지급은 유료에서만 가동한다.
*   **Q2. 무료 상점 트래픽 및 알림톡 비용 폭증 리스크**:
    *   A2. 알림톡 발송은 반드시 **카드 등록 상점**에 한정하며, 50건 초과 시 발송 기능을 자동 잠금한다. 월 주문 100건 이상 상점은 무료 혜택에서 배제하는 다운그레이드 필터를 가동한다.
*   **Q3. 기존 유료 고객의 역차별 불만**:
    *   A3. 무료 플랜을 "깊이가 얕은 입문 패키지"로 명문화하고, 유료 전용 기능(자동화, 브랜딩 제거, 고급 위젯 21종 등)의 경계를 자연스럽게 대조 표시한다.

### 📖-5. No-Gos (범위 배제)

*   **디자인 세부 커스터마이징**: 폰트, 상세 컬러 팔레트, 다중 레이아웃 및 페이지별 위젯 조합 제외.
*   **포인트 정교화**: 조건별 차등 보상 지급, 이벤트 보너스, 쿠폰 연동 등 제외.
*   **프라이싱 패키지 전면 개편**: 행동 데이터 수집만 완료해 두고 정식 가격 리디자인은 다음 단계로 보류.

### 📖-6. Definition of Done

*   **동작 기준**: 6주 배팅 종료 시점까지 무료 플랜 설치 → 온보딩 완료 → 소셜/스태프 리뷰 1건 이상 연결 → 인스타 피드 위젯 노출까지 한 사이클을 완수한 상점이 **최소 20~30개** 이상이어야 한다.
*   **작성률 개선**: 무료 상점의 평균 리뷰 작성률이 사용 전 대비 **+20%** 이상 향상됨을 데이터 상으로 검증해야 한다.
